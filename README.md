# Sistema de Reserva de Hotéis

Este repositório contém o código-fonte e a documentação do projeto de um **Sistema de Reserva de Hotéis**, desenvolvido como requisito para a disciplina de **Prática Profissional em Análise e Desenvolvimento de Sistemas** da Universidade Presbiteriana Mackenzie.

## Sobre o Projeto

O objetivo principal deste projeto é aplicar os conceitos de engenharia de software, especificamente o **Processo Unificado (PU)**, para desenvolver uma solução de software funcional e bem documentada. O sistema visa simplificar e otimizar o processo de reserva de hospedagens, atendendo às necessidades de hóspedes e da administração hoteleira.

## Estrutura do Repositório

- **/docs**: Contém toda a documentação da fase de Concepção, incluindo a análise de requisitos, diagramas de casos de uso e o documento de visão do projeto.
- **/src**: Destinado ao código-fonte da aplicação (a ser desenvolvido nas próximas fases).
- **/tests**: Conterá os casos de teste e scripts de validação.

## Metodologia

O desenvolvimento segue um modelo iterativo e incremental, alinhado com as quatro fases do Processo Unificado:

1.  **Concepção**: Definição do escopo e visão do projeto. (Fase atual)
2.  **Elaboração**: Detalhamento dos requisitos e arquitetura do sistema.
3.  **Construção**: Implementação e codificação do software.
4.  **Transição**: Implantação e entrega final do produto.
